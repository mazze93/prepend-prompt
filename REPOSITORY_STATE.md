### Description

Transforms ordinary user queries into persuasion-enhanced prompts based on Robert Cialdini's influence principles. By selecting a principle at runtime, the Shortcut automatically wraps the input in a structured template and sends it to ChatGPT.

---

## Project Purpose

This project implements a **psychological persuasion framework** for AI interactions by:

1. **Applying Cialdini's Six Principles of Influence**:
   - Reciprocity: Leveraging obligation and value exchange
   - Commitment and Consistency: Referencing established positions
   - Social Proof: Using consensus and popular behavior
   - Authority: Establishing expertise and credibility
   - Liking: Building rapport and common ground
   - Scarcity: Emphasizing uniqueness and urgency

2. **Runtime Principle Selection**: Users choose which influence principle to apply
3. **Automatic Prompt Enhancement**: Wraps user queries in principle-specific templates
4. **Direct ChatGPT Integration**: Sends enhanced prompts to ChatGPT API

## Installation
### Option 1: Download the Shortcut (Ready to Use)
1. Navigate to the [Releases page](../../releases)
2. Download the `.shortcut` file for your platform.
   - **iOS**: Ensure Shortcuts app version is 14.0 or higher.
   - **macOS**: Ensure compatibility with Mac running 12.0 or later.
3. Double-click or tap to automatically open the file using the Apple Shortcuts app.
4. Follow the guided setup for API key configuration.

### Option 2: Manual Setup
1. Open the Shortcuts app on your iOS/macOS device.
2. Select ‚ÄúCreate New Shortcut‚Äù from the menu.

4. Manually enter the API key during variable setup to integrate with ChatGPT.

### Note:
- We highly recommend keeping your ChatGPT API key private during manual setup. Your key enables the magic of enhanced conversations but must remain secure.
- All shortcut modifications or updates get easier once Option **direct install** grows with release improvements on version rollout.
